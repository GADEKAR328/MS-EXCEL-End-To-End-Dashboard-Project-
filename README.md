# 📊 Super Store Sales Dashboard | Excel Project  
**By Yogesh Gadekar**

---

## 🚀 Project Overview  

This project presents an **interactive Sales Dashboard built using Microsoft Excel**, transforming raw Superstore data into meaningful business insights.

The objective is to demonstrate **data cleaning, transformation, and visualization skills** using Excel tools like **Power Query, Pivot Tables, and Dashboard Design**.

---

## 🎯 Key Objectives  

- Convert raw sales data into structured format  
- Perform data cleaning & preprocessing  
- Analyze sales trends across regions and years  
- Build an interactive and dynamic dashboard  
- Extract actionable business insights  

---

## 🛠️ Tools & Technologies Used  

- 📊 Microsoft Excel  
- 🔄 Power Query (Data Cleaning & Transformation)  
- 📈 Pivot Tables & Pivot Charts  
- 🎨 Dashboard Design (UI/UX in Excel)  

---

## 🔄 Project Workflow  

### 1️⃣ Data Cleaning (Power Query)
- Removed duplicates and null values  
- Standardized column formats  
- Fixed inconsistent data entries  
- Prepared structured dataset  

### 2️⃣ Data Transformation
- Created calculated fields  
- Organized data for analysis  
- Prepared dataset for reporting  

### 3️⃣ Data Analysis (Pivot Tables)
- Region-wise sales analysis  
- Category and segment performance  
- Monthly trends and order patterns  

### 4️⃣ Dashboard Creation
- Designed interactive layout  
- Added slicers (Region & Year filters)  
- Built charts for data visualization

## 🧠 Key Insights  

- Consumer segment contributes the highest order quantity  
- West and East regions show strong performance  
- Peak sales observed in November and December  
- Standard Class shipping dominates sales share  
- Office Supplies category leads in transactions
